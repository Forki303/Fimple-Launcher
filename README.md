# Fimple Launcher

**Fimple Launcher** — простой и удобный лаунчер для Minecraft.

С ним вы сможете:  
- Выбирать и запускать любую версию Minecraft.  
- Настраивать лаунчер под себя с помощью простого интерфейса.  
- Играть без лишних сложностей и долгих настроек.  

**Установка:**  
1. **Обычная версия:**  
   - Поместите `.exe` файл в удобное место и запустите его.  
2. **Версия 5.0:**  
   - Скачайте архив с версией 5.0.  
   - Поместите архив в удобное место и распакуйте его.  
   - После распаковки запустите `.exe` файл из распакованной папки.  

Если возникли проблемы — создайте [дискуссию](https://github.com/Forki303/Fimple-Launcher/discussions) на GitHub, и я постараюсь помочь.

# Fimple Launcher

**Fimple Launcher** — a simple and convenient launcher for Minecraft.

With it, you can:  
- Choose and launch any Minecraft version.  
- Customize the launcher through a simple interface.  
- Play Minecraft quickly without extra hassle.

**Installation:**  
1. **Regular version:**  
   - Place the `.exe` file in a convenient location and run it.  
2. **Version 5.0:**  
   - Download the archive containing version 5.0.  
   - Place the archive in a convenient location and extract it.  
   - After extraction, run the `.exe` file from the extracted folder.  

If you encounter any problems — create a [discussion](https://github.com/Forki303/Fimple-Launcher/discussions) on GitHub, and I will try to help.
