# Fimple Launcher

**Fimple Launcher** — простой и удобный лаунчер для Minecraft.

С ним вы сможете:  
- Выбирать и запускать любую версию Minecraft.  
- Настраивать лаунчер под себя с помощью простого интерфейса.  
- Играть без лишних сложностей и долгих настроек.  

**Установка:**  
Поместите `.exe` файл в удобное место и запустите его.

# Fimple Launcher

**Fimple Launcher** — a simple and convenient launcher for Minecraft.

With it, you can:  
- Choose and launch any Minecraft version.  
- Customize the launcher through a simple interface.  
- Play Minecraft quickly without extra hassle.

**Installation:**  
Place the `.exe` file in a convenient location and run it.
